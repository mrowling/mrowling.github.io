Multi-Tenant Database Server

Go MySQL Protocol SQLite Docker

A MySQL-compatible multi-tenant database server built in Go, providing database isolation per tenant using SQLite backends. Designed to facilitate multi-tenancy features for testing purposes. Features include MySQL wire protocol implementation, RESTful API, session management, and comprehensive Docker-based testing.

  • MySQL protocol compatibility with per-tenant database isolation
  • Dynamic database creation and tenant routing
  • HTTP REST API for database management
  • Comprehensive test suite with Docker integration
  • Query logging and auditing capabilities

Homelab Infrastructure

Kubernetes K3s Ansible Proxmox ArgoCD GitOps

Production-grade homelab running a highly available K3s cluster on Proxmox with GitOps via ArgoCD. Features automated infrastructure provisioning, comprehensive monitoring stack, and self-hosted services including Nextcloud, Paperless-NGX, Vaultwarden, and media management.

  • HA K3s cluster with 3 masters and 3 workers on Proxmox
  • Automated VM provisioning with Ansible
  • GitOps deployment pipeline with ArgoCD
  • Longhorn distributed storage and Velero backups
  • Self-hosted services: Nextcloud, Paperless-NGX, Vaultwarden, Mealie, n8n
  • Monitoring with Grafana Agent and observability stack
